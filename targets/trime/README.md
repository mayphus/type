# Trime Skin Review

This folder keeps the current Trime review inputs and the minimal patch for the WeChat green skin.

- `trime.yaml` is the stock Trime theme reference pulled from Trime's shared folder.
- `../../profiles/yuanshu/customer/微信·绿最终版v3.cskin` is the original Yuanshu skin package.
- `trime.custom.yaml` patches the default Trime theme.
- `standard.custom.yaml` carries the same patch for Trime installs where the active theme is `standard`.

Prefer `trime.custom.yaml` over a copied full theme until layout changes are truly needed.
